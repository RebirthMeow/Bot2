# Off-Mesh Connection Generation Spec
## OpenJK Navmesh Generator — Passes 1–3

This document specifies a post-geometry detection pass for the OpenJK navmesh generator that identifies and bakes off-mesh connections for three movement types: ledge drops, basic jumps, and wallrun-ascend. The pass runs after navmesh polygons are built but **before** `dtCreateNavMeshData` is called, and outputs connection data into `dtNavMeshCreateParams`.

---
